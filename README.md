# Swift Supply Solutions Inc. — Website

Static HTML website for Swift Supply Solutions Inc. (swiftsupplysolutions.ca / sssol.ca)

## Pages

| File | Page |
|------|------|
| `index.html` | Home |
| `services.html` | Services |
| `about.html` | About |
| `case-studies.html` | Case Studies |
| `insights.html` | Insights Index |
| `contact.html` | Contact |
| `insight-hidden-cost-print-mail.html` | Insight: Hidden Cost of Print & Mail |
| `insight-why-print-mail-still-matters.html` | Insight: Why Print Still Matters |
| `404.html` | 404 Error Page |

## Notes

- All assets (logos, fonts) are embedded as base64 — no external dependencies
- To add a new Insight article: duplicate an existing article HTML file, update content, add card to `insights.html`
- Site is designed to deploy to sssol.ca (staging) and migrate to swiftsupplysolutions.ca (production)
